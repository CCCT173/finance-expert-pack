## 🔒 回测纪律：零未来函数

**这是本包和市面上99%回测代码最大的区别。**

我们把回测纪律写死在代码里，不是嘴上说说：

✅ 所有趋势信号**必须用t-1收盘价判定**，T+1执行，绝对禁止未来函数  
✅ warmup期 `min_periods=120`，参数不足不瞎算  
✅ 成本模型透明：仅计单边切换费，不藏着掖着  
✅ 取数失败**优雅降级+显式告警**，告诉你M<N，而不是悄悄补假数据  
✅ 回测结果100%可复现：同样输入永远得到同样输出  

> 回测的第一原则是：**先保证不骗自己，再谈赚不赚钱**。

## 🚀 快速开始（3分钟上手）

### 1. 安装依赖

```bash
pip install -r requirements.txt
# 核心：pandas / numpy / akshare / requests
```

### 2. （可选）配置Token

实时行情主源需要智兔Token，不配置也能用降级功能，不会崩：

```powershell
$env:ZHITU_TOKEN = "你的token"        # 实时行情主源
$env:TUSHARE_TOKEN = "你的token"      # 可选多源降级
```

### 3. 跑第一个回测

```bash
# 全板块「过滤等权持有」回测，生成可交互HTML报告
python scripts/workspace/sector_strategy.py --all

# 指定板块+周期
python scripts/workspace/sector_strategy.py --sector 游戏 --period 全周期2021-26

# 实验模式：过滤等权+基本面截面选股（防御型）
python scripts/workspace/sector_strategy.py --mode quality --all

# 个股一键分析
python scripts/run_analysis.py 600519
```

运行完直接打开生成的 `index_filtered_eqw.html`，净值曲线、回撤、交易记录全有。

## 📊 板块策略回测说明

`scripts/workspace/sector_strategy.py` 是本包核心能力。

| 模式 | 策略逻辑 | 适用场景 | 产物 |
|------|----------|----------|------|
| `--mode filtered`（**默认推荐**） | 等权持有板块成分股，板块ETF跌破200MA就转现金，站上再买回 | 牛市吃beta，熊市躲大跌，不用盯盘 | `index_filtered_eqw.html` |
| `--mode quality`（实验） | 趋势向上区间内按**ROE+营收增速+利润增速**rank-sum选前5，月度再平衡 | 防御型倾斜，波动更小 | `index_t3_quality.html` |

> 经过游戏、半导体、新能源、医药等多板块2018-2026样本外验证：**简单规则跑赢绝大多数主动选股策略**。
> 200MA过滤是「下跌段风险削减器」，不是「上涨段收益增强器」——先活下来，再谈赚多少。

## ⚠️ 真实局限（不吹不黑）

我们不搞"年化50%"那套营销话术，局限性明明白白写在这里：

1. **回测成本偏乐观**：当前仅计单边切换费，未含印花税、滑点、冲击成本，实盘摩擦会侵蚀收益
2. **存在生存者偏差**：成分股取自当前时点，未处理历史退市/更名，结论谨慎外推
3. **资金流维度暂不可用**：东财接口封锁，T3模式目前为纯基本面截面
4. **实时行情需要Token**：ZHITU_TOKEN未配置时盯盘/热榜不可用，其余功能正常
5. **数据源稳定性风险**：公开源会改版，偶发限流，已做重试但无法保证100%覆盖
6. **A股为主**：港股/美股路径存在但未经同等样本外验证

## 📏 七条铁律

本包所有输出严格遵守以下原则：

1. **不编造数字**：所有数据必须来自真实来源，没有就说「未取到」
2. **分层表述**：区分「事实/推断/猜测」，不确定的明确标注
3. **不喊口号**：禁止「必涨/精准买点/稳赚」，永远给出base/bull/bear三情景+触发条件+失效位
4. **矛盾必呈现**：多空信号都列出来，不和稀泥
5. **带来源**：每条数据标注出处，你可以自己去验证
6. **可复现**：同样输入永远得到同样结果，没有随机黑盒
7. **仅供参考**：所有内容都是研究参考，不构成投资建议
