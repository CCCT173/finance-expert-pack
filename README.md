## 🔒 回测纪律：敢上实盘的硬标准
**这是本包和市面上99%回测代码的本质区别。**

我们把回测纪律写死在代码里，不是嘴上说说：

✅ **真实交易成本**：佣金万2.5+印花税千1+过户费十万1+滑点千1，扣除全部费用后的收益才是真收益  
✅ **零未来函数**：所有信号必须用t-1收盘价判定，T+1开盘成交，用未来价格算收益的把戏一律不玩  
✅ **真实交易规则**：涨停买不进、跌停卖不出、停牌自动跳过、ST/*ST/次新/低流动性股自动过滤  
✅ **参数不足不瞎算**：warmup期min_periods=120，数据不够直接空仓，不凑结果  
✅ **取数失败明说**：数据缺失显式告警告诉你M<N，绝不偷偷补假数据凑曲线  
✅ **100%可复现**：同样输入永远得到同样输出，没有随机黑盒，结果可以自己验证

> 回测的第一原则：**先保证不骗自己，再谈赚不赚钱。**

## ⚠️ 真实局限（不吹不黑）
我们不搞"年化50%"那套营销话术，局限性明明白白写在这里：

1. **存在生存者偏差**：成分股取自当前时点，未处理历史退市/更名，结论谨慎外推
2. **资金流维度有限**：东财接口封锁，T3质量模式目前为纯基本面截面
3. **数据源稳定性**：公开源会随网站改版波动，偶发限流，已做重试降级但无法保证100%覆盖
4. **A股为主**：港股/美股路径存在但未经同等样本外验证

## 📏 七条铁律
本包所有输出严格遵守，绝不越界：

1. **不编造数字**：所有数据来自真实来源，没有就说「未取到」
2. **分层表述**：区分「事实/推断/猜测」，不确定的明确标注

6. **可复现**：同样输入永远得到同样结果，没有随机黑盒
