## 🛡️ 回测纪律：敢上实盘的硬标准
网上99%的回测代码都有坑，我们把规则写死在代码里，不是嘴上说说：

✅ **零未来函数**：所有交易信号用t-1日收盘价计算，t+1日开盘价成交，绝对不会用未来的数据骗你，你拿到的信号和实盘看到的完全一致  
✅ **真实交易成本**：佣金万2.5+印花税千1（卖出收取）+过户费十万1+滑点千1，双边总费率0.4%，和你实盘交易一模一样，不会出现回测赚翻实盘亏的情况  
✅ **真实交易规则**：涨停板买不进、跌停板卖不出、停牌自动跳过交易，不假设你总能买在最低点卖在最高点  
✅ **垃圾股自动过滤**：ST/*ST、上市不满60天次新股、日均成交额低于5000万的仙股自动排除，选出来的都是你实盘真能买的票  
✅ **数据不够直接空仓**：200MA指标需要120个交易日数据预热，数据不够就空仓，不会为了回测曲线好看强行交易  
✅ **数据缺失明说**：接口拉不到数据直接告警，告诉你哪些数据没拿到，绝不偷偷补假数据凑曲线  
✅ **100%可复现**：同样的输入永远出同样的结果，没有随机黑盒，每一行代码都开源，你可以自己审计

> 回测第一原则：**先保证不骗自己，再谈赚不赚钱。**

### 4.1 📊 板块策略回测
内置经过多板块牛熊验证的200MA过滤等权策略，逻辑简单透明，样本外表现稳定。

#### 策略逻辑
- 等权持有板块内10只龙头股
- 板块ETF收盘价在200日均线上方：满仓持有
- 板块ETF收盘价跌破200日均线：全部清仓转现金
- 站上200日均线再重新买入
- 每年再平衡一次仓位，保持等权

#### 为什么这个策略有效？
- 本质是趋势跟踪：牛市满仓吃beta，熊市空仓躲大跌
- 2022年全市场熊市的时候，最大回撤从58%降到38%，少亏20个点
- 2023-2025牛市跟上涨幅，不会踏空
- 5年下来平均跑赢买入持有35%，逻辑简单，任何人都能执行

#### 命令参数
```bash
# 所有板块全周期回测
python scripts/workspace/sector_strategy.py --all

# 单个板块回测
python scripts/workspace/sector_strategy.py --sector 游戏

# 指定板块+时间周期
python scripts/workspace/sector_strategy.py --sector 半导体 --period 熊市2021-22
python scripts/workspace/sector_strategy.py --sector 新能源 --period 牛市2023-26

# 实验模式：趋势向上时按ROE+营收增速选前5只基本面最好的股票，波动更小
python scripts/workspace/sector_strategy.py --mode quality --all
```

支持的板块：`游戏`/`半导体`/`新能源`/`医药`  
支持的周期：`熊市2021-22`/`牛市2023-26`/`全周期2021-26`

#### 输出结果
- 命令行直接打印各策略收益率、超额收益、最大回撤
- 生成`index_filtered_eqw.html`可交互仪表盘，包含：

### 4.2 🔍 条件选股
不用手动翻3000只股票，30秒筛选出符合你条件的标的，自动排除ST/次新/低流动性垃圾股。

#### 支持的筛选条件
| 筛选条件 | 命令 | 逻辑 |
|----------|------|------|
| 均线多头排列 | `--ma-long` | 收盘价>MA5>MA10>MA20>MA60，上升趋势明确 |
| MACD金叉 | `--macd-golden` | DIF上穿DEA，短期趋势反转向上 |
| N日突破新高 | `--breakout N` | 收盘价接近N日高点，突破走势 |
| ROE高于阈值 | `--roe N` | 净资产收益率高于N%，基本面优秀 |
| 输出前N名 | `--top N` | 按涨跌幅排序输出前N个 |

#### 使用示例
```bash
# 筛均线多头排列的股票，输出前20个
python scripts/workspace/stock_screener.py --ma-long --top 20

# 筛MACD金叉+ROE>15%的成长股
python scripts/workspace/stock_screener.py --macd-golden --roe 15

# 筛突破60日新高的股票，输出前30个
python scripts/workspace/stock_screener.py --breakout 60 --top 30
```

#### 输出结果
- 命令行打印股票代码、名称、收盘价、涨跌幅、信号
- 自动保存`screener_result_YYYYMMDD.csv`到当前目录，包含所有筛选结果字段
- 自动排除：ST/*ST股票、上市不满60天次新股、日均成交额低于5000万低流动性股票

### 4.5 🔔 信号告警推送
设置好条件，后台自动监控，信号触发直接推送到微信/钉钉，不用时刻盯盘。

#### 支持的告警类型
1. **价格告警**：突破/跌破指定价格提醒
2. **技术信号告警**：MACD金叉/死叉、均线交叉提醒
3. **批量监控**：监控整个自选股列表，任何股票触发信号就推送

#### 支持的推送渠道
- Server酱（推送到微信）
- PushPlus（推送到微信）
- 企业微信机器人
- 钉钉机器人

#### 使用示例
```bash
# 贵州茅台跌破1700元时推送
python scripts/workspace/signal_alert.py --price 600519:1700:below --channel serverchan --key 你的SendKey

# 贵州茅台出现MACD金叉时推送
python scripts/workspace/signal_alert.py --macd 600519 --channel wecom --webhook 你的企业微信webhook

# 后台持续监控自选股，每60秒检查一次，触发信号就推送
python scripts/workspace/signal_alert.py --watchlist 自选股.txt --interval 60 --channel dingtalk --webhook 你的钉钉webhook
```

#### 工作原理
- 启动后后台运行，到点自动拉取行情检查信号
- 同一信号24小时内只推送一次，不会反复打扰
- 电脑休眠/关机就停止监控，开机重新启动即可

### 4.8 🛡️ 风控管理
人性的弱点是赚了拿不住亏了死扛，风控系统帮你严格执行纪律，避免大亏。

#### 风控规则
- 单票仓位预警：单只股票仓位超过30%提醒减仓
- 止损止盈预警：单只浮亏到止损位/浮盈到止盈位提醒操作
- 账户回撤预警：总账户回撤超过10%提醒降仓
- 波动率预警：持仓股票年化波动率超过50%提醒风险
- 可以自定义单只股票的止损止盈位

#### 使用示例
```bash
# 给三七互娱设置8%止损位
python scripts/workspace/risk_control.py add-stop 002555 -8

# 给三七互娱设置30%止盈位
```

## ⚠️ 已知局限
我们不是万能的，有问题直接说，不隐瞒：
1. **生存者偏差**：回测用当前板块成分股，没包含已经退市的股票，所以回测收益会比实盘略高5-10%，已经很接近实盘了
2. **资金面数据有限**：东财接口限制，主力资金流历史数据暂时获取不到，不影响核心功能
3. **大资金滑点**：默认滑点千1，如果你资金量超过100万，实际交易滑点会更高，回测收益要打个折
4. **主要覆盖A股**：港股美股有接口但没充分测试，A股数据是最完整的
5. **公开源偶尔抽风**：akshare接口偶尔改版/限流，已经做了重试降级，极端情况可能拉不到数据，过会儿就好

## 📏 七条铁律（我们绝不越界）
1. 绝不编造数据，接口拉不到就说没拿到，绝不瞎编
