# 每日财经热榜 / 投资资讯 工作流

> 本 skill 不内置独立脚本抓取热榜（避免与平台能力重复）。当需要"每日财经热榜 / 投资资讯"时，
> 按本流程获取并结构化输出。两条可选路径：**A. 平台联网能力（默认、可移植）** 或 **B. 私有运行时（OpenClaw/JPRX）**。

---

## 路径 A：走平台联网能力（推荐，无私有依赖）

直接使用对话环境的联网搜索 / 网页抓取能力，按下列来源汇总：

| 类别 | 来源 | 抓取要点 |
|------|------|----------|
| 宏观 / 政策 | 财联社、华尔街见闻、第一财经、央行/证监会官网 | 当日要闻、政策发布、流动性操作 |
| 市场快讯 | 东方财富快讯、同花顺快讯 | 盘面异动、涨跌停、资金切换 |
| 个股 / 行业 | 雪球、同花顺、公告（巨潮/东财） | 个股重大公告、研报观点、行业动态 |
| 全球 | Investing.com、Yahoo Finance | 美股/商品/汇率/美债收益率 |

**标准输出格式**（供下游消费）：

```json
{
  "date": "YYYY-MM-DD",
  "macro": [{"title": "...", "source": "...", "impact": "利好/利空/中性"}],
  "market_brief": "...一句话盘面总结...",
  "hot_sectors": ["AI", "半导体", "黄金"],
  "watchlist_news": [{"code": "600519", "title": "..."}],
  "risks": ["...需要关注的尾部风险..."]
}
```

**纪律**：标题/数据必须带来源；无法核实的标"未证实"；禁止编造具体数字或引语。

---

## 路径 B：私有运行时（OpenClaw/JPRX 已部署时）

若环境已配置 `AUTH_GATEWAY_PORT` 且 `PROSEARCH_SCRIPT` 指向可用脚本，可直接调用：

```powershell
python scripts/workspace/news_radar.py
```

该脚本经网关查询个股公告/研报（neodata）并调用元宝搜索抓取实时新闻，输出结构化发现。
网关/搜索不可用时脚本优雅降级（返回空或提示），不会崩溃。

---

## 频率与时效

- 盘中（9:30–15:00）：聚焦快讯与异动，时效以分钟计。
- 盘前/盘后：聚焦隔夜海外、政策、公告与研报，时效以小时/日计。
- 同一会话内已拉取且时间戳未过旧（< 1h）可复用，避免重复抓取。
