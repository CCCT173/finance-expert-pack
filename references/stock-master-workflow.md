## 一、硬性规则（所有场景通用）

1. **数据必须来自真实来源**（akshare/yfinance/web search/官方披露），禁止编造数字
2. **区分事实、推断、猜测**，不确定结论标注"推断"或"情景"
3. **禁止"必涨/精准买点/稳赚"**，用 base/bull/bear 三情景 + 触发条件 + 失效位
4. **矛盾必须呈现**，不准和稀泥——估值冲突、信号分歧本身就是信息
5. **缺失不编造**，找不到数据时标记"未取到"，不猜测
6. **每条数据带来源**，web search 兜底时标记 fallback
7. **不构成投资建议**，所有输出为研究参考

## 三、个股深度分析

### 3.1 分析框架（Step 0-8 + 技术面 + 资金面）

**Step 0：身份确认**
- 股票代码/名称/市场（A/H/U）/行业/概念板块
- 若名称模糊，列出 Top3 候选让用户确认

**Step 1：基本面速览**
- 最新价/涨跌幅/市值/PE(TTM)/PB/52周高低
- 行业排名（PE/PB/ROE 分位）

**Step 2：5年财务趋势**
- ROE/营收/净利/毛利率/资产负债率/现金流
- 画出趋势方向（上升/平稳/下降/波动）

**Step 3：同行对比**
- 同行业 Top5 对比：PE/PB/ROE/营收增速/净利增速
- 该股在行业中的定位（龙头/中游/尾部）

**Step 4：主营构成与产业链**
- 分产品/分地区营收占比
- 上游供应商集中度 + 下游客户集中度
- 在产业链中的议价能力

**Step 5：资金面**
- 北向资金近20日趋势
- 融资融券余额变化
- 主力资金近5日净流入/流出
- 机构持仓变化（近8季）
- 股东户数变动方向

**Step 6：龙虎榜与游资**
- 近期是否上榜
- 买入/卖出席位
- 游资识别（知名席位匹配）

**Step 7：治理与风险**
- 股权质押比例
- 高管增减持
- ST/违规/限售解禁
- 十大股东变动

**Step 8：估值判断**
- PE/PB 历史分位（5年）
- DCF 简算（FCF + 增长率 + WACC）
- 同行 PE 均值对比
- 结论：高估/合理/低估，附置信度

**Step 9：技术面**
- 日K/周K趋势 + MA5/10/20/60 位置
- 成交量变化（放量/缩量）
- MACD/KDJ/RSI 信号
- 支撑位/阻力位
- 相对强度（vs 大盘）

**Step 10：事件与催化**
- 近期新闻/公告/研报
- 业绩预告/快报
- 行业政策/催化日历

### 3.2 陷阱检测清单

逐项检查，命中即标红：

| 陷阱类型 | 检测方法 |
|----------|----------|
| 财务造假嫌疑 | 经营现金流 vs 净利润长期背离、应收暴增、存货异常 |
| 大股东掏空 | 关联交易占比高、质押率>50%、频繁增减持 |
| 估值陷阱 | PE低但ROE持续下滑、周期股在盈利峰值 |
| 流动性陷阱 | 日成交额<5000万、股东户数骤降 |
| 概念炒作 | 营收无实质变化但股价暴涨、概念板块蹭热点 |
| 商誉雷 | 商誉/净资产>30%、收购标的业绩变脸 |

## 四、每日市场简报

### 4.1 数据采集清单

按优先级采集，缺项标记"未取到"：

1. **地缘/事件风险**：冲突/制裁/贸易/外交/峰会/央行会议 → 多空分类 → 板块映射
2. **海外市场**：标普/纳指/道指/美10Y/美元指数/USD-CNH/金/铜/油/VIX
3. **A股盘面**：上证/沪深300/创业板/科创50/成交额/涨跌比/涨停跌停/北向南向
4. **主力资金**：板块/概念主力净流入/大单/ETF份额变化/大宗
5. **技术面**：主要指数日/周K线 + MA5/10/20/60 + 量价 + MACD/KDJ/RSI
6. **政策/宏观**：最新政策/央行操作/财政/地产/消费/科技
7. **财报/估值**：最新季报/年报/营收利润趋势/估值/拥挤度
8. **催化日历**：财报/分红/指数再平衡/央行会议/行业会议

### 4.2 综合研判流程

1. **事件风险图**：列出重大地缘/政策/峰会催化 → 多空分类 → 板块映射
2. **盘面确认/否定**：盘面是否验证事件叙事？矛盾处必须解释
3. **市场风险评分**：0-100 分 + 一句话理由
4. **主线识别**：当前主线 / 下一主线候选 / 潜伏候选
5. **板块评分**：政策+宏观+财报+资金+技术 五维打分
6. **区分故事驱动 vs 财报确认**：故事驱动降级处理

### 4.3 输出模板

```

| 事件 | 多空 | 影响板块 | 状态 |

### 主线与候选
- 当前主线: ...
- 下一主线候选: ...
- 潜伏候选: ...

### 重点板块评分
| 板块 | 政策 | 宏观 | 财报 | 资金 | 技术 | 综合 |

### 操作建议
[进攻/防守/观望] + Top 1-3 主题 + 不买/暂缓理由

### 下一关注
1. ...

```

## 五、资金轮动分析

### 5.1 分析流程

**1. 确定轮动范围**
- 市场：US / China/HK / Taiwan / 全球
- 周期：日/周 / 1-3月 / 6-18月
- 单位：板块 / 主题 / ETF / 个股
- 目标：探测下一方向 / 防守持仓 / 候选对比 / 轮动地图

**2. 判断市场体制**
从资金环境出发：
- 流动性：利率/央行资产负债表/信用利差/美元/长端利率
- 宏观冲击：CPI/油价/地缘/衰退信号/日元套利
- 风险偏好：广度/等权vs市值权/高beta/小盘/IPO
- AI/商业验证：云增长/AI capex/模型公司收入/企业采纳

归类体制：全面流动性牛市 / 结构性牛市 / 估值重置 / 盈利重置 / 逻辑重置 / 宏观逻辑重置

**3. 绘制资金来源与去向表**

| 来源 | 去向 | 原因 | 证据 | 风险 |
|------|------|------|------|------|

**4. 归类轮动逻辑**
- `稀缺性`：产能短缺/定价权/交期延长/产能分配/积压
- `升级`：架构/效率变革迫使新组件/材料/电力/封装/测试/网络
- `远期期权`：边缘AI/物理AI/机器人/自动驾驶/AI-RAN

**5. 回调压力测试**
- `估值杀`：价格跑在证据前面，商业逻辑完好
- `盈利杀`：增长/指引低于预期，信心需新数据
- `逻辑杀`：战略角色/客户资格/架构/市场结构破裂

**6. 候选赛道评分（1-5分）**
- 需求证据 / 供给紧张 / 盈利转化时点 / 相对强度确认 / 拥挤/估值风险(反向) / 失效清晰度

**7. 输出**

快速回答：轮动结论 + Top去向 + 资金来源 + 确认证据 + 失效信号

完整报告：体制快照 + 板块热力图 + 稀缺/升级/远期矩阵 + 个股观察名单 + 回调分类 + 催化日历 + 轮出触发条件

## 六、供应链瓶颈挖掘

### 6.1 核心模型

把市场当物理系统，而非行情馈送：

```
超级趋势 → 供应链/栈映射 → 找最窄物理/资格/产能约束 → 交叉验证 → 先输出方向 → 再给候选标的
```

默认行为：
- **不**从报股票名开始
- **不**问AI"该买什么"
- **要**用AI扩展搜索半径、映射依赖、总结财报、交叉验证瓶颈、生成证伪测试

### 6.2 双透镜

**Serenity 透镜**（找被忽视的瓶颈点）：
- 一个缺失零件可以阻滞更大的AI建设
- 价值常在二阶/三阶瓶颈，而非显性龙头
- 市值错配重要：小供应商能延迟大需求波 → 定价错误可能很大

寻找：供给集中 / 认证周期长 / 不可替代 / 产能满载 / 管理层用"sole source/primary source/qualification/ramp/demand>supply"

**Crux 透镜**（全栈+仓位逻辑）：
- 光/AI基建是一个栈
- 每层因不同原因、不同时间线获得报酬
- 组合构建与选股同等重要
- 权重应跟踪执行确定性

### 6.3 工作流（L1-L5 分层对话）

**L1：板块诊断** — 确认超级趋势
- 什么需求波在扩张？什么物理建设意味着什么？
- 哪些组件必须随之扩展？哪些已是共识和拥挤？
- 用具体机器/系统锚定（不是"AI计算"，而是"GB300 NVL72机架"）

**L2：栈映射** — 6-9层
```
终端需求 → 网络/系统 → 模块/引擎/子系统 → 器件/芯片/激光器/光学 → 测试/良率/可靠性 → 代工/封装 → 外延/设备 → 材料/衬底/特种输入
```
每层问：运的是什么？谁在这里赚钱？什么解锁下一层？瓶颈是产能/资格/热/良率/工具/材料？

**L3：证据链** — 三桶验证
1. 公司证据：财报电话会/投资者展示/客户供应商提及/指引
2. 行业证据：行业媒体/报告/产能/交期/部署新闻
3. 跨链证据：多家公司从不同位置描述同一压力点

**L4：方向输出**（默认输出层）
- 下一值得跟踪的方向 + 为什么是现在 + 什么确认它 + 什么打破它 + 上下游谁先感受

**L5：小市值深挖**（仅用户追问时）
- 3-7个标的，按角色分组：最安全执行者 / 纯瓶颈供应商 / 更便宜的二阶受益者 / 早期期权/颠覆者
- 每个标明：栈中角色 / 为什么属于这里 / 哪些证据是实的 / 还需确认什么 / 主要风险

### 6.4 瓶颈评分

| 维度 | 评分要点 |
|------|----------|
| 供给集中度 | 垄断/双寡头/寡头/竞争 |
| 可替代性 | 直接替代/部分变通/无近期替代 |
| 扩产周期 | 月/年/结构性受限 |
| 认证壁垒 | 客户验证/工艺诀窍/IP/可靠性/监管 |
| 定价权 | 明确涨价/积压/受限分配/利润率拐点 |

## 七、美国政治交易信号

### 7.1 数据源路径

**国会**：Capitol Trades / Quiver Quant / Unusual Whales / House Clerk PTR / Senate eFD

**行政分支**：Open Cabinet / TrumpTrades / Trump Tracker / OGE 公开披露

### 7.2 工作流

1. **定义猎取范围**：人物/角色/ticker/公司/板块/日期/交易类型/来源
2. **采集交易记录**：官方/成员/角色/议院或机构/党派州/委员会/ticker/交易类型/交易日期/申报日期/金额范围/账户/来源URL
3. **验证与标准化**：确认ticker身份/纠正二手源错误/标记ETF/基金/债券/RSU/自动计划/信托/配偶账户
4. **映射角色-板块重叠**：委员会/机构/政策管辖 → ticker业务/客户/供应商/监管敞口
5. **构建时间链**：交易日期 → 申报日期 → 公开催化日期 → 价格反应
6. **信号质量排名**
7. **报告**

### 7.3 信号评分（100分制）

| 组件 | 分值 | 检查内容 |
|------|------|----------|
| 多官员集群 | 0-20 | 30-60天内多人买卖同一ticker/板块 |
| 角色相关性 | 0-25 | 委员会/机构/政策管辖直接重叠 |
| 时机vs催化 | 0-20 | 公开新闻前交易/催化前申报/低点买入 |
| 交易质量 | 0-15 | 主动个股/期权/大额/重复行为 |
| 披露新鲜度 | 0-10 | 最近申报且仍可操作 |
| 反向卖出预警 | 0-10 | 相关官员在不利政策/盈利/监管风险前卖出 |

解读：75-100 高信号 / 55-74 中信号 / 35-54 低信号 / <35 弱/无信号

### 7.4 注意事项

- 不称"内幕交易/非法"，用"政策信息优势/角色-板块重叠/冲突风险信号"
- 金额是范围非精确值
- 区分主动交易 vs ETF/RSU/自动/合规/信托
- 交易日期 ≠ 申报日期

## 八、基金筛选

### 8.1 筛选规则

**被动基金**：跟踪指数质量 / AUM / 流动性/成交量 / 跟踪误差 / 费率 / 溢价折价 / 集中度 / 指数是否匹配主题

**主动基金**：经理任期 / 风格一致性 / 持仓集中度 / 换手率 / 回撤控制 / 超额收益来源 / 容量压力 / 近期持仓是否仍符论点

**QDII/海外基金**：额度/申购状态 / 溢价风险 / 汇率敞口 / 当地市场假期 / 底层市场估值 / 结算延迟

**ETF**：底层板块/指数K线 + ETF自身流动性/溢价折价/盘中趋势/量变确认

**战术思路**：入场触发 / 失效条件 / 仓位角色 / 预期持有期 / 风险情景

### 8.2 输出模板

| 基金代码 | 名称 | 类型 | 最新净值 | 费率 | 论点 | 风险 | 操作建议 |

## 十、输出纪律

1. **先给结论**：一句话定调 + Top 1-3 要点
2. **只给最相关数据**，附来源链接
3. **用表格**呈现基金候选/板块评分/交易记录
4. **包含"不买/暂缓"理由**，不只说正面
5. **结尾给下一步监控**：具体数据点/政策事件/量价触发/资金流确认
6. **矛盾呈现**：估值冲突/信号分歧 → 把冲突本身当信息
7. **行动标签**：`积累研究` / `持有观察` / `等回调` / `轮动候选` / `回避至证据改善` — 禁止绝对买卖指令
